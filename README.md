# TUBAF_Fruehjahrsakademie2026
Der Code für die Experimente zur Frühjahrsakademie Mathematik 2026 an der TU Bergakademie Freiberg.

## Code

- **01_Einfache_Klassifizierung.ipynb** Zeigt Klassifizierung mit keras auf Fashin-MNIST Daten.
- **02_Bild_Klassifizierung.ipynb** Zeigt wie Bilder effizient mit einem CNN klassifiziert werden können.
- **99_Perceptron_MNIST.ipynb** Die adversarial Attacke auf ein Perceptron.
